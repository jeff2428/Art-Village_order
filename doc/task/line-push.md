# LINE 推播模塊任務

## 目標

訂單成立後，透過 LINE Messaging API 將完整訂單明細推播至店家內部 LINE 群組。

## 子任務

- [ ] 設定 LINE Channel Access Token
- [ ] 設定店家 LINE 群組 ID
- [ ] 建立訂單推播訊息格式
- [ ] 訊息包含訂位人姓名與電話
- [ ] 訊息包含人數、日期與時間
- [ ] 訊息包含餐點與客製化細節
- [ ] 呼叫 LINE Messaging API
- [ ] 記錄推播成功或失敗結果

## 驗收

- [ ] 訂單成立後店家 LINE 群組收到通知
- [ ] 推播內容包含完整訂單明細
- [ ] 推播失敗不會造成訂單資料遺失
