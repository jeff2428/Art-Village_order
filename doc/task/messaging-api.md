# Messaging API 整合任務

## 目標

整合 LINE Messaging API，將訂單明細傳送至店家內部 LINE 群組。

## 子任務

- [ ] 建立 LINE Messaging API Channel
- [ ] 取得 Channel Access Token
- [ ] 取得店家內部 LINE 群組 ID
- [ ] 將 token 存放於 GAS Script Properties
- [ ] 建立推播 API 呼叫函式
- [ ] 建立推播錯誤處理

## 驗收

- [ ] GAS 可成功呼叫 LINE Messaging API
- [ ] 店家群組可收到測試推播
